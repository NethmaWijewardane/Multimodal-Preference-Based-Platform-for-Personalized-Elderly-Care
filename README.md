# Elderly Caregiver Matching System

## Project Description
This project is developed as a Final Year Project for the BSc (Hons) Computer Science degree.

The system provides a platform where elderly individuals can find suitable caregivers based on specific requirements such as language skills, patience level, and preferred activities. Caregivers can accept or decline service requests, and elderly users can provide feedback and ratings after receiving services.

## Main Features
- Separate authentication for Elderly and Caregivers
- Caregiver profile management
- Requirement-based caregiver search and filtering
- Accept / Decline service requests
- Feedback and rating system
- Secure backend with database integration

## Technologies (Planned)
- Frontend: React
- Backend: Node.js with Express
- Database: MongoDB
